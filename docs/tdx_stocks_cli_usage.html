tdx-stocks CLI 使用说明图

推荐入口是 `sync`、`data`、`audit`、`query`；`--dry-run` 只看不改，`--json` 适合脚本，旧命令保留为兼容别名。

怎么开始

  • • 先跑 `tdx-stocks --help` 看完整入口。
  • • 想快速了解命令结构，用 `tdx-stocks help-summary`。
  • • 不确定是否要改数据时，先用 `tdx-stocks sync --dry-run`。

最常用的三类动作

  • • `sync`：一键同步和重建。
  • • `audit verify`：检查复权一致性。
  • • `query price` / `query sql`：只读查看行情和自定义查询。

脚本与兼容性

  • • `--json` 适合自动化脚本和 CI。
  • • 写操作会加锁，防止并发破坏缓存。
  • • 旧命令仍可执行，但会提示迁移到新入口。
